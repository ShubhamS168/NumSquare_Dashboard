Welcome to the Data Analysis Dashboard

This project demonstrates how to build a web-based data analysis tool using Flask, SQLite, and Matplotlib. You can input numbers, compute their squares, store results in a database, and visualize them as a chart.

How to use

Enter a number below and click "Calculate" to compute its square. The result will be stored in the database. You can view all results on the "View Results" page and visualize them on the "View Chart" page.
